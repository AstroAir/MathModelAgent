## 📋 使用条款详解

### 1. 个人使用

**定义**: 个人用户在非商业环境下使用

**允许**:
- 下载和安装软件
- 用于个人学习和研究
- 参加数学建模竞赛
- 分享使用经验

**不允许**:
- 将软件用于商业项目
- 提供付费服务
- 销售或出租软件

### 2. 学术使用

**定义**: 教育机构和研究机构使用

**允许**:
- 用于教学活动
- 用于学术研究
- 发表学术论文（需注明来源）
- 课堂演示

**要求**:
- 在论文或报告中注明使用了 MathModelAgent
- 引用格式：
  ```
  MathModelAgent: An AI-powered Mathematical Modeling System
  GitHub: https://github.com/jihe520/MathModelAgent
  ```

### 3. 商业使用

**定义**: 任何以营利为目的的使用

**需要授权的情况**:
- 在商业项目中使用
- 提供基于本软件的付费服务
- 集成到商业产品中
- 为客户提供建模服务

**授权流程**:
1. 联系作者说明使用场景
2. 协商授权条款和费用
3. 签署商业许可协议
4. 获得授权后方可使用

### 4. 二次开发

**允许**:
- Fork 项目进行修改
- 提交 Pull Request
- 创建衍生项目（需开源）

**要求**:
- 保留原始版权声明
- 开源您的修改（不可闭源）
- 注明基于 MathModelAgent 开发
- 使用相同或兼容的许可证

**不允许**:
- 闭源分发修改版本
- 移除版权信息
- 声称为原创作品
